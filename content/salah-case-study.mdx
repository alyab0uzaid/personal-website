# Case Study: Salah

A prayer times and Qibla app with tracking, notifications, and tasbih. Clean, minimal design with location-based calculation methods.

## Overview

Salah helps Muslims stay on track with daily prayers by providing accurate prayer times, Qibla direction, and a built-in tasbih counter. The app uses the device location to calculate prayer times based on the user's local methods.

## The Problem

Many prayer time apps feel cluttered or outdated. Users need reliable, accurate times with a simple, focused interface that doesn't distract from worship.

## The Solution

- **Location-based prayer times**: Accurate calculations using Adhan library with support for multiple calculation methods
- **Qibla direction**: Compass integration for finding the direction of the Kaaba
- **Tasbih counter**: Built-in counter for dhikr and remembrance
- **Notifications**: Timely reminders for each prayer
- **Clean, minimal design**: Focused experience without visual clutter

## Tech Stack

- React Native, Expo
- JavaScript
- Adhan (prayer time calculations)
- Expo Location
- Expo Notifications

## Outcome

A polished prayer app available on the App Store, built with a focus on reliability, accuracy, and a calm user experience.
