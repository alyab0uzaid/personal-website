# Case Study: RabbitHole Chrome Extension

A Chrome extension that integrates Wikipedia content and visualizes your exploration journey across webpages.

## Overview

RabbitHole helps users discover and explore related topics without leaving the page they're on. It surfaces Wikipedia content in context and tracks your exploration path, turning casual browsing into a more intentional learning experience.

## The Problem

When researching online, users often jump between tabs and lose track of their exploration path. Finding related context without disrupting your flow was difficult.

## The Solution

- **Inline Wikipedia integration**: View summaries and links to related topics without leaving the current page
- **Visual exploration map**: See your journey across topics with GoJS-powered diagrams
- **Chrome Extension API**: Seamless integration into the browsing experience

## Technologies

- JavaScript, HTML, CSS
- GoJS for graph visualization
- Chrome Extension API
- Wikipedia API

## Outcome

A polished extension that makes knowledge exploration more fluid and visual. Built over two months with a focus on smooth UX and reliable Wikipedia integration.
