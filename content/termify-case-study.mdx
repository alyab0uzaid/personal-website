# Case Study: Termify

Termify provides a unique tactile experience for Spotify users, where they can retrieve top tracks, save playlists, and log out from the terminal interface.

## Overview

A Node.js CLI that integrates with the Spotify API, built with Express and Pug for a polished terminal experience.

## Tech Stack

- Node.js, Express
- Spotify API
- Pug templates
