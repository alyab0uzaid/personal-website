# Case Study: The Country Game

A geography word game where players name a country for each letter of the alphabet, with optional timers. Supports solo, local multiplayer, and real-time online multiplayer with room codes.

## Overview

The Country Game brings a familiar word-game format to geography, making it easy to play with friends both locally and online.

## Tech Stack

- React, TypeScript, Vite, Tailwind CSS
- Supabase for backend
- PartyKit for real-time multiplayer
- Zod for validation
